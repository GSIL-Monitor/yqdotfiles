Pausing folders

If you need to pause processing for a particular folder, you can do so from the preference pane interface. This is useful, for example, in cases where you want to preview your rules before allowing Hazel to run them.

To pause processing for a folder:

  1. Select the folder you want to pause.
  2. From the action pull-down (), select "Pause Rules"
  3. The folder will show a paused icon to indicate its paused state.

To resume processing for a folder:

  1. Select the folder you want to pause.
  2. From the action pull-down (), select "Resume Rules"
  3. The folder will no longer show a paused icon.

While a folder is paused, Hazel will not trigger processing of any rules for that folder. Note that any processing already in progress will continue running until completion.
