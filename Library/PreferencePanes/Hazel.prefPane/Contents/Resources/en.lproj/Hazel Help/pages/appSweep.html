App Sweep

While software on the Mac is mostly self-contained, there are still support files that may get created. Your preferences for that application, for instance, are stored in a separate file. When your throw the app away, these support files remain on your system, creating clutter.

With App Sweep, you can fully uninstall software by just throwing it in the trash. Hazel will notice when applications and other software are thrown away and search for their support files. If it finds any, Hazel will offer to throw these support files as well, thus fully uninstalling the software.

Note that if the software in question has its own uninstaller, it is recommended that you use that to uninstall the software. While App Sweep can deal with most software installations, certain apps may require special steps that only it knows about. That goes for Hazel itself. While App Sweep can be used to uninstall it, the built-in uninstaller (read about uninstalling Hazel) will do a better job as it also will terminate its background processes for you. In general, if you installed the application by dragging it to your disk (as opposed to running a special installer program or package), then it is a good candidate for App Sweep.

Multi-user App Sweep

App Sweep can also handle uninstalling across users on the same machine. If you check the "Enable multi-user sweep" option, Hazel will keep track of software you have thrown away. When other users on that machine log in, they will be given the option to likewise throw away support files as if they had thrown away the software themselves.

Note that this will only work across users that have enabled this option. Users opting out of this will not be given a chance to throw away support files for software other users have thrown away nor will other users be allowed to throw away support files for software they throw away themselves.
