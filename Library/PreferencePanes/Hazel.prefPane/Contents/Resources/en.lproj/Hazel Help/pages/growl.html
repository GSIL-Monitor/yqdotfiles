Using Growl

Growl is a free notification system. With Growl installed, you can get notifications on your Desktop about what Hazel is doing.

Hazel has four types of notifications that it sends out:

  • File Events
    These notifications occur when a file is changed using Hazel's built-in actions. Significant actions such as moving, copying, archiving and unarchiving are considered file events.
  • Trash Events
    These events relate to Trash events, usually when Hazel deletes something permanently from the Trash.
  • Errors
    This covers any errors that occur while processing rules.
  • Custom Messages
    Custom messages are sent by the "Send Growl notification" action in rules. The messages themselves can be defined by you.

To enable Growl support:

  1. Make sure Growl is installed. Download it from their website and follow the instructions there.
  2. In Growl's preferences, navigate to the "Applications" tabs.
  3. Select Hazel from the list.
  4. Enable or disable the different types of notifications as you see fit.

In addition to the standard messages that Hazel can send out, you can have your rules send out custom messages defined by you. To do this, just use the "Send Growl notification" action. Consult the Rule Reference for more details.

Clicking on Growl notifications

For many alerts, clicking on a Growl notification will show you in Finder the file to which the notification pertains. If you would like to dismiss the notification window without showing the file, hold down the Option key while clicking on the notification.
