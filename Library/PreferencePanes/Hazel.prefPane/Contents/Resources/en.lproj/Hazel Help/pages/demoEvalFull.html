Demo mode vs Evaluation mode vs Full

When you first download and install Hazel, if you do not register it, it will run in Evaluation mode. In this mode, all features except rule export are enabled, allowing you to try Hazel out. This mode lasts for 14 days after which time, it will go into Demo mode. In this mode, Hazel will only monitor one folder and only two rules can be active. Import and export functions will be unavailable as well.

To enjoy the full featureset of Hazel, you need to purchase a full license. When fully registered, Hazel will run with all features unlocked without any expiration.
