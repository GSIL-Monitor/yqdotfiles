Loading the sample rules

Hazel has a set of sample rules you can load at any time. These rules provide basic filling capability as well as serving as a starting point for your own rules.

When you start Hazel for the first time, you are given the opportunity to load these rules. If you opted not to load the rules at that time and you need to load them into a new folder, you can still load them yourself.

To load the sample rules:

  1. Select the folder you want to add the rules to.
  2. From the action pull-down (), select "Load Sample Rules."
  3. The sample rules are loaded in an active state. Enable the rules you want by clicking on the checkbox next to the rule.
