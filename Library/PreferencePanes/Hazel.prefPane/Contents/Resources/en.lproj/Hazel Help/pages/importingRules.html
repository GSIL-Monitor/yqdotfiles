Importing rules

You can add rules to the current folder from a file. You can use this feature to restore rules from a previous export or to load in rules you have downloaded from others.

To import rules:

  1. Select the folder you want to import to.
  2. Select the rule in the rule list after which you want the rules added.
  3. From the action pull-down (), select "Import Rules…"
  4. Select the file containing the rules you want to import and click "Open"
  5. The rules you imported will be loaded in an inactive state. Enable the rules you want by clicking on the checkbox under the "Active" column in the rule list.
