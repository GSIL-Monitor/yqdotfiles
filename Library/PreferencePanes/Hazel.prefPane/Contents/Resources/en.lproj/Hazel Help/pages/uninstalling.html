Uninstalling Hazel

Uninstalling Hazel will result in the removal of all settings and files related to Hazel. Note that this will remove the license file registered with Hazel so make sure you have a copy of the original license file somewhere safe (like the original mail). This procedure will require a restart of System Preferences.

If you installed Hazel machine-wide (you selected "Install for all users" when installing), you should uninstall Hazel from an admin account otherwise it may fail.

To uninstall Hazel:

  1. Navigate to the "Info" pane.
  2. Click on the "Uninstall" button in the lower left.

Hazel will quit and restart System Preferences when it has completed.
