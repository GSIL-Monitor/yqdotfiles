Installing Hazel

Before installation, make sure your system meets the following system requirement:

  • Mac running OS X 10.10 (Yosemite) or above.

Also, you should check the Noodlesoft web site to make sure you have the latest version.

To install Hazel:

  1. Double-click on the disk image file you downloaded.
  2. Click "Accept" to accept the license terms. If you do not agree, then click "Do not accept." Note that doing so will not allow you to install Hazel.
  3. Double-click the Hazel icon. It will launch System Preferences which will, in turn, ask if you would like to install Hazel systemwide or for the current user. If in doubt, select "current user."
  4. Eject the installation disk image. You can do this by clicking on the "Eject" icon next to the disk image icon in the Finder's sidebar.
  5. Drag the disk image to the Trash.
  6. [optional] You can click on "Check Updates" when you are done to make sure you are up to date. The automatic update system will allow you to automatically download and install updates.
