What's New in Hazel 4

Hazel 4 adds new features to make creating rules easier

Live Rule Preview

See how your rule matches up as you're editing it. Just tell Hazel which file to preview against and it will show you which conditions match and which ones don't. You can dig deeper by seeing the values of any attributes you are using in the conditions.

Rule Sync

You can now create even more powerful rules. With nested conditions, condition targets & AppleScript and shellscripts in conditions, the sky's the limit.

Syncing Rules

Smart Folders

Hazel can now monitor Smart Folders as it does regular folders. Rules will work like they do in regular folders, except that anything requiring going into subfolders or matching against sub-files is not allowed.

Rule Search

You can now search/filter your rule list. Hazel will find things based on anything you entered in the name, the notes or any of the rule's fields.
