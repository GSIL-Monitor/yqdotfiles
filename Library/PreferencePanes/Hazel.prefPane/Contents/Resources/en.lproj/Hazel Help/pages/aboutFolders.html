About Folders


What folders can Hazel organize?

Hazel can be used to organize most any folder, but the best candidates are ones that tend to collect files from the outside, such as where your browser downloads files or where Mail puts attachments.

Note that not all folders are good candidates. Some programs (such as P2P and BitTorrent clients) download files over a long period of time. Hazel needs clues to know that the program has finished downloading otherwise it may end up moving the file out from underneath the other program. In general, Hazel needs special support for the program to prevent any conflicts.

Hazel can also monitor Smart Folders. You can specify rules to operate on the contents of a Smart Folder as you would a regular folder. Note, though, that there are limitations. You cannot have rules that match on, or go into, sub-files and folders.

Supported Programs and Quick Folders

When selecting a folder, Hazel provides a "Quick Folder" list providing quick access to the folders used by various programs that Hazel supports. Here's the current list:

  • Safari: The folder where you set Safari to download to, settable in its preferences.
  • Mail: When you open an attachment, Mail stores a copy of it here. You can set this location in Mail's preferences.
  • Firefox: In Firefox, you have the option to have it automatically download everything to a folder. This setting as well as the download location are settable in Firefox's preferences.
  • Opera: Opera allows you to set the folder where all downloads go. This is settable in its preferences.
  • Camino: Camino has a single folder where all downloads go which is settable in its Preferences. Note that this is the same folder that Safari uses so when you change one, you change the other.
  • Flock: Like Firefox, you have the option to have Flock automatically download everything to a folder. This setting as well as the download location are settable in Flock's preferences.
  • Transmission: To prevent Hazel from intefering with Transmission downloads, you must set Transmission to keep incomplete files in a separate folder. When this is set in Transmission's preferences, the Quick Folder link will point to Transmission's "Default location" which is where completed downloads are kept.

What if my favorite app is not supported?

If your app is not supported, there are a couple things you can do:

  • Tell us about it! We are constantly looking to improve Hazel so if we know what you are using, the more likely we will support it in the future.
  • Some programs have an option to move the file somewhere else when it's done. For instance, Azureus has such options under "File: Completion Moving". In such a case, you can configure the program to move files to a folder that Hazel monitors. You should not have Hazel monitor the folder that the program actually downloads to because of the aforementioned likelihood of Hazel processing it prematurely.
