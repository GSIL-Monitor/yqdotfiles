Running rules

Normally, Hazel runs rules automatically as needed. Occasionally, though, you may need to force rules to run.

To force rules to run:

  1. Select the folder whose rules you want to run.
  2. From the action pull-down (), select "Run Rules Now".

Alternatively, if you enabled the menu bar icon:

  1. Click on Hazel's menu bar icon.
  2. Select the "Run Rules" item.
  3. In the submenu that appears, select the folder whose rules you want to run.

If you want to see what Hazel is doing, you can view the log by clicking on the "View Log" button in the "Info" pane or you can set up Growl and receive visual notifications on your Desktop.
