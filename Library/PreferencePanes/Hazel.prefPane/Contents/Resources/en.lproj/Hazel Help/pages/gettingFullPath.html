Getting the full path of a folder

The folder list only shows the folder name. You can easily get the full path (all of the folders above this one).

How to get the full path of a folder:

  1. Hover the mouse over the folder (do not click).
  2. The full path should be displayed as a tool tip.
  3. If you would like to see the folder in Finder, right-click or control-click on the folder and select "Reveal in Finder".
