Viewing the logs

If your rules are not running as expected, you can view the logs to see what actually happened. Here you can see if Hazel encountered any errors when running your rules.

Note that it helps to preview your rules beforehand. See Previewing rules for more details.

To view the logs:

  1. Go to the "Info" section.
  2. Click "View Log".

Alternatively, if you enabled the menu bar icon:

  1. Click on Hazel's menu bar icon.
  2. Select "View Log".

Either method above will launch the Console application and display the logs. You can use the search field to search for any particular files you are interested in.
