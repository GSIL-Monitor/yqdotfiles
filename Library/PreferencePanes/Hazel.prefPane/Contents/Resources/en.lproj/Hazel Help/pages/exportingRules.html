Exporting rules

You can save your rules to a file. These files can be used as backup or they can be traded with others.

To export rules:

  1. Select the folder whose rules you want to export. Keep in mind that all rules for the folder will be exported.
  2. From the action pull-down (), select "Export Rules…"
  3. Select the folder where you want to save the rules. If you wish, you can change the name of the file.
  4. Click "Save".

Alternatively, you can export rules for all your folders at once.

To export all rules:

  1. From the action pull-down (), select "Export All…"
  2. Select the folder where you want to save the rules. A file will be created for each folder's rulesets.
  3. Click "Save".
