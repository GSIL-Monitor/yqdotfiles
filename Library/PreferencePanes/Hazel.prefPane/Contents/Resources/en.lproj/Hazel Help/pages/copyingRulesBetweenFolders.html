Copying (or moving) rules between folders

In addition to importing and exporting whole rule lists, you can copy or move individual rules from one folder to another.

To copy a rule to another folder:

  1. Click and drag the rule to copy.
  2. Drag the rule to the folder you would like to copy the rule to.
  3. If you have "spring-loaded folders" enabled in Finder's preferences, the folder will be selected if you hold the rule over the folder.
  4. Drop the rule on the folder. This will result in the rule being added to the end of the rule list for that folder. Alternatively, if you have "spring-loaded folders" enabled, you can drag the rule back to the rule list to drop it in a specific position.
  5. You can likewise move a rule by holding down the command key during the drag and drop. The green plus (+) icon will disappear in such a case.
